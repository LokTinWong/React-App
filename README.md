# React App

A full-stack app built with React following [this YouTube tutorial](https://youtu.be/SqcY0GlETPk?si=XK9CdDiTEdddHAU2).

## Setup Instructions

1. Type `node -v` in your terminal, and ensure that you have **Node** version 16 or newer installed. If not, go to [https://nodejs.org/en/download/current](https://nodejs.org/en/download/current) and get your own version of **Node**.

2. Type `npm -v` in your terminal to check if you have **npm** installed. To install the latest version of **npm**, type the following in your terminal:

```
npm install -g npm
```

3. Clone this GitHub repo.

4. In your terminal, navigate to the directory of the repo folder.

5. When you want to launch the app, type `npm run dev` in the terminal.

6. Click the **localhost** link. It would look something like [http://localhost:5173/](http://localhost:5173/).

7. Type `q` in the terminal if you want to quit hosting, or type `h` for more shortcuts.
